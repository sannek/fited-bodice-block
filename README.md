# danckaerts-bodice

## Install

```bash
npm install --save danckaerts-bodice
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'danckaerts-bodice'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
